# Create Timetable

The **Create Timetable** feature allows you to generate and manage class schedules efficiently. You can set start and end times, define timeslot durations, and assign subjects to each day.

---

## How to Create a Timetable

1. **Go to:**  
   **Timetable** ‚Üí **Create Timetable**

2. **Set Timetable Parameters:**
   - **Starting Time:** Set the start time for the school day (e.g., 07:00 AM)
   - **Ending Time:** Set the end time for the school day (e.g., 01:30 PM)
   - **Timeslot Duration:** Enter the duration for each period in minutes (e.g., 30)
   - **Medium:** Select the medium (e.g., English, Hindi) from the dropdown

3. **Generate Timetable:**
   - Click **Generate** to create the timetable grid based on your parameters

4. **Assign Subjects:**
   - For each day and timeslot, assign the appropriate subject (e.g., Mathematics, English, EVS, etc.)
   - You can add multiple subjects per day as needed

5. **Edit or Delete Entries:**
   - Use the **edit** (‚úèÔ∏è) and **delete** (üóëÔ∏è) buttons in the Action column to modify or remove timetable entries

6. **Save or Export:**
   - Use the available options to save or export the timetable as needed

## Best Practices
- Double-check timeslot durations to avoid overlaps
- Assign subjects and teachers carefully for each slot
- Regularly review and update the timetable as needed

---

## Troubleshooting
- **Cannot Generate Timetable?**  
  Ensure all required fields (start time, end time, duration, medium) are filled
- **Entries Not Saving?**  
  Check for missing or invalid data in the timetable grid
