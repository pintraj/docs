# Theme Color

This page explains how to set and customize the primary, secondary, background, and text colors for your School CRM system.

## How to Configure Theme Colors

1. Go to **Web Settings > Theme Color** in the sidebar.
2. Adjust the following color options as needed:
   - **Primary Color**
   - **Secondary Color**
   - **Primary Background Colour**
   - **Text Secondary Colour**
   - **Primary Hover Colour**
3. Enter the desired color codes or use the color picker.
4. Save your changes to apply the new theme.
