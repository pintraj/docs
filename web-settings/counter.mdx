# Counter

This page explains how to update the Counter section, which displays key statistics or achievements on your website.

## How to Update Counter

1. Go to **Web Settings > Counter** in the sidebar.
2. Add or edit the counters as needed (e.g., number of students, teachers, awards).
3. Save your changes.
