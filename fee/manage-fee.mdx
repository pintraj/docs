# Manage Fees

The **Manage Fees** feature allows you to create, edit, and list different types of fees for students. You can set up compulsory, optional, and installment fees for selected classes and sessions.

---

## How to Manage Fees

1. **Go to:**  
   **Fees** ‚Üí **Manage Fees**
2. **Create Fees:**
   - **Prefix Name:** Enter a prefix for the fee (required)
   - **Classes:** Select one or more classes (required)
   - Use **Select All** to assign the fee to all classes
3. **Set Compulsory Fees:**
   - Select the fee type and enter the amount
   - Add due date and due charges (percentage or amount)
   - Click **Add New Data** to add more compulsory fees
4. **Set Fees Installment:**
   - Enable or disable installment option
5. **Set Optional Fees:**
   - Add optional fees as needed (note: optional fees do not support due charges or installment)
6. **Submit:**
   - Click **Submit** to save the fee structure
   - Use **Reset** to clear the form
7. **List Fees:**
   - View all created fees in the table below
   - Edit or delete fees as needed

## Best Practices
- Set up all fee types at the start of the academic year
- Use clear names and descriptions for each fee
- Regularly review and update fees as needed

---

## Troubleshooting
- **Cannot Add Fee?**  
  Ensure all required fields are filled
- **Fee Not Showing?**  
  Check the selected class and session filters
