# Fees Transaction Logs

The **Fees Transaction Logs** feature allows you to view all online fee payment transactions. You can filter by payment status, month, and session year, and see details for each transaction.

---

## How to View Fees Transaction Logs

1. **Go to:**  
   **Fees** â†’ **Fees Transaction Logs**
2. **Filter Transactions:**
   - Select payment status, month, and session year to filter the logs
3. **View Transaction Table:**
   - The table displays:
     - User
     - Amount
     - Payment Gateway
     - Payment Status (e.g., Pending, Completed)
     - Date
