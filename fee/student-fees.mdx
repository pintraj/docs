# Student Fees

The **Student Fees** feature allows you to view, filter, and update the payment status of student fees. You can track compulsory and optional fees, payment methods, and fee status for each student.

---

## How to Manage Student Fees

1. **Go to:**  
   **Fees** â†’ **Student Fees**
2. **Filter Fees:**
   - Select the session year, fee type, and class section to filter the list
   - Filter by payment status (e.g., Unpaid, Partial Paid, Over Due)
3. **View Student Fees Table:**
   - The table displays:
     - Student Name
     - Class
     - Compulsory Fees
     - Optional Fees
     - Payment Method
     - Fees Status (e.g., Partial Paid, Over Due)
     - Date
     - Paid Amount
     - Action (e.g., update payment, view receipt)
4. **Update Fee Status:**
   - Click the action button to update payment status or view/download receipts

## Best Practices
- Regularly review student fee status to ensure timely payments
- Use filters to quickly find unpaid or overdue fees
- Download receipts for record-keeping

---

## Troubleshooting
- **Fees Not Showing?**  
  Ensure the correct filters are applied
- **Cannot Update Status?**  
  Check user permissions or contact support
