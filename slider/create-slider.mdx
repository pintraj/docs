# Create New Slider

The **Create New Slider** feature allows you to add promotional or informational banners to your app or web portal. You can upload an image, add a link, and select where the slider will appear.

---

## How to Create a New Slider

1. **Go to:**  
   **Sliders** â†’ **Create New Slider**
2. **Upload Image:**
   - Click **Upload** and select an image file from your computer (required)
3. **Add Link:**
   - Enter a URL to link the slider to a web page (optional)
4. **Select Type:**
   - Choose where the slider will appear: **App**, **Web**, or **Both**
5. **Submit:**
   - Click **Submit** to add the slider
   - Use **Reset** to clear the form

---

## Best Practices
- Use high-quality images for better visual impact
- Ensure links are correct and relevant
- Choose the appropriate type to target your audience

---

## Troubleshooting
- **Cannot Upload Image?**  
  Check file format and size
- **Slider Not Showing?**  
  Ensure the type and link are set correctly
