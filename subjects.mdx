## Managing Subjects

School CRM provides an intuitive interface to create and manage academic subjects. You can easily add new subjects and associate them with specific language mediums.

### Adding New Subjects

Follow these steps to add a new subject:

1. Navigate to the Subjects section
2. Click on "Add Subject" button
3. Enter subject details and select the language medium
4. Upload a subject image (optional)
5. Click "Save" to create the subject

![Subject Management](/images/create-subjects.png)

### Subject Images

You can enhance subject presentation by adding custom images. Create professional subject images using these free tools:

- [Canva](https://www.canva.com/) - Professional design tool with education templates
- [BeFunky](https://www.befunky.com/create/) - Easy-to-use photo editor and graphic designer

### Managing Existing Subjects

The system allows you to:
- Edit subject details
- Update subject images
- Delete subjects when needed
![Subject Management](/images/list-of-subjects.png)
<Note>
Make sure to review subject details before deletion as this action cannot be undone.
</Note>
