# Manage Exam

The **Manage Exam** feature allows you to view, edit, and delete offline exams. You can update exam details or remove exams that are no longer needed.

---

## How to Edit or Delete an Exam

1. **Go to:**  
   **Offline Exam** → **Manage Exam**
2. **View Exams:**
   - The table lists all exams with columns for name, session, classes, and action
3. **Edit an Exam:**
   - Click the **edit** icon (✏️) in the Action column
   - Update the exam name, session, classes, or description as needed
   - Click **Submit** to save changes
4. **Delete an Exam:**
   - Click the **delete** icon (🗑️) in the Action column
   - Confirm the deletion if prompted

## Best Practices
- Regularly review and update exams to keep records accurate
- Remove outdated or irrelevant exams promptly

---

## Troubleshooting
- **Cannot Edit or Delete?**  
  Ensure you have the necessary permissions
- **Changes Not Saving?**  
  Check for required fields and try again
