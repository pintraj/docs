# Create Exam

The **Create Exam** feature allows you to set up new offline exams for selected classes and session years. You can specify the exam name, session, classes, and provide a description.

---

## How to Create an Exam

1. **Go to:**  
   **Offline Exam** â†’ **Create Exam**
2. **Enter Exam Details:**
   - **Exam Name:** Enter the name of the exam (required)
   - **Session Years:** Select the academic session (required)
   - **Classes:** Select one or more classes to assign the exam (required)
   - **Exam Description:** (Optional) Add a description for the exam
   - Use **Select All** to assign the exam to all classes
3. **Submit:**
   - Click **Submit** to create the exam
   - Use **Reset** to clear the form

---

## Best Practices
- Use clear, descriptive exam names
- Assign exams to the correct classes and session
- Add a description for clarity

---

## Troubleshooting
- **Cannot Submit?**  
  Ensure all required fields are filled
- **Exam Not Visible?**  
  Check the selected session and class filters
