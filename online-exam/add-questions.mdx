# Add Questions

The **Add Questions** feature allows you to create and manage questions for online exams. You can add questions, options, correct answers, and images for each question.

---

## How to Add Questions to an Online Exam

1. **Go to:**  
   **Online Exam** ‚Üí **Add Questions**
2. **Select Exam Details:**
   - **Class Section:** Select the relevant class section (required)
   - **Subject:** Select the subject for the exam (required)
3. **Enter Question Details:**
   - **Question:** Enter the question text (required)
   - **Add Option:** Click **Add Option** to add answer choices (A, B, C, D, etc.)
   - **Answer:** Select or enter the correct answer (required)
   - **Image:** (Optional) Upload an image related to the question
   - **Note:** (Optional) Add any additional notes for the question
4. **Submit:**
   - Click **Submit** to save the question

## Best Practices
- Use clear and concise language for questions and options
- Add images for visual questions when needed
- Double-check the correct answer before submitting

---

## Troubleshooting
- **Cannot Add Question?**  
  Ensure all required fields are filled
- **Image Not Uploading?**  
  Check file format and size
