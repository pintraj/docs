# Create Payroll

The **Create Payroll** feature allows you to generate payroll records for staff for a specific month and year. You can enter payroll details and submit them for processing.

---

## How to Create Payroll

1. **Go to:**  
   **Payroll** â†’ **Create Payroll**
2. **Select Payroll Period:**
   - Choose the month and year for which you want to create payroll (required)
3. **Enter Payroll Details:**
   - Enter or review payroll details for each staff member as required
4. **Submit Payroll:**
   - Click **Submit** to generate payroll for the selected period
   - Use **Reset** to clear the form

---

## Best Practices
- Ensure all staff details are accurate before submitting payroll
- Generate payroll at the end of each month

---

## Troubleshooting
- **Cannot Submit Payroll?**  
  Ensure all required fields are filled
- **Payroll Not Generating?**  
  Refresh the page or check for errors
