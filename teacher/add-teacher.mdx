# Add Teacher

The **Add Teacher** feature allows you to create a new teacher profile in School CRM, including personal details, contact information, qualifications, and salary information.

---

## How to Add a Teacher

1. **Go to:**  
   **Teacher** → **Manage Teacher**

2. **Fill in the Teacher Details:**
   - **Email** (required)
   - **First Name** (required)
   - **Last Name** (required)
   - **Gender** (required): Select Male or Female
   - **Mobile** (required)
   - **Date of Birth** (required)
   - **Image**: Click **Upload** to add a profile image (308px × 338px, JPG/PNG)
   - **Qualification** (required)
   - **Current Address** (required)
   - **Permanent Address** (required)
   - **Salary** (required)
   - **Joining Date** (required)
   - **Status** (required): Select Active or Inactive

3. **Allowances and Deductions:**
   - Add any applicable **Allowances** by selecting the type and clicking **Add New Allowances**
   - Add any **Deductions** by selecting the type and clicking **Add New Deduction**

4. **Submit:**
   - Click **Submit** to save the new teacher profile
   - Use **Reset** to clear the form

---

## Best Practices
- Ensure all required fields are filled accurately
- Use a clear, professional photo for the teacher profile
- Double-check salary, joining date, and contact details before submitting

---

## Troubleshooting
- **Cannot Submit?**  
  Check for missing required fields or invalid data
- **Image Not Uploading?**  
  Ensure the image meets the size and format requirements
