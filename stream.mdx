## What are Academic Streams?

Academic streams represent the different educational pathways or specializations that students can choose from. Each stream typically has its own set of subjects, curriculum requirements, and career focus areas.

### Common Academic Streams

- **Science Stream**: Focuses on Mathematics, Physics, Chemistry, Biology
- **Commerce Stream**: Emphasizes Business Studies, Accountancy, Economics
- **Arts Stream**: Covers Literature, History, Geography, Political Science

## Managing Existing Streams

### View All Streams

The stream management page displays all created streams in a table format:

| Stream Name | Code | Status | Students | Actions |
|-------------|------|--------|----------|---------|
| Science | SCI | Active | 150 | Edit, Delete |
| Commerce | COM | Active | 120 | Edit, Delete |
| Arts | ART | Active | 80 | Edit, Delete |

### Edit Stream Details

1. Click the **Edit** button next to the stream you want to modify
2. Update the required information
3. Click **Save Changes**

<Callout type="info">
  **Note**: Changes to stream configuration will affect all classes and students associated with that stream.
</Callout>

### Delete Stream

<Callout type="warning">
  **Important**: Deleting a stream will affect all associated classes and students. Consider deactivating the stream instead of deleting it.
</Callout>

1. Click the **Delete** button next to the stream
2. Confirm the deletion in the popup dialog
3. The stream will be permanently removed from the system



### Naming Conventions

- Use clear, descriptive names (e.g., "Science with Mathematics" instead of "Sci-Math")
- Create consistent codes (e.g., "SCI-MATH", "COM-ACC")
- Include grade level indicators if needed (e.g., "Science-11", "Commerce-12")
