# Create Announcement

The **Create Announcement** feature allows you to share important updates, events, or information with selected class sections. You can add a title, description, upload files, include a URL, and target specific classes.

---

## How to Create an Announcement

1. **Go to:**  
   **Announcement** → **Create Announcement**
2. **Enter Announcement Details:**
   - **Title:** Enter the announcement title (required)
   - **Description:** Add a brief description (optional)
   - **Files:** Upload an image or document file (max 1 MB; required file types: image or document)
   - **Add URL:** (Optional) Add a link for more information
   - **Class Sections:** Select one or more class sections to receive the announcement (required)
   - Use **Select All** to target all class sections
3. **Submit:**
   - Click **Submit** to publish the announcement
   - Use **Reset** to clear the form

## Best Practices
- Use clear, concise titles and descriptions
- Ensure files are within the allowed size and type
- Target announcements to relevant class sections

---

## Troubleshooting
- **Cannot Upload File?**  
  Check file type and size (must be < 1 MB)
- **Announcement Not Visible?**  
  Ensure the correct class sections are selected
