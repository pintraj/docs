# View Attendance

The **View Attendance** feature allows you to check the daily attendance status of students for a selected class and date. You can see each student's admission number, roll number, name, and attendance type (e.g., Present, Absent, Holiday).

---

## How to View Student Attendance

1. **Go to:**  
   **Attendance** â†’ **View Attendance**
2. **Select Class Section:**
   - Use the dropdown to choose the relevant class section (e.g., 1 A - English)
3. **Select Date:**
   - Pick the date for which you want to view attendance
4. **View Attendance Table:**
   - The table displays:
     - Admission Number
     - Roll Number
     - Name
     - Type (e.g., Present, Absent, Holiday)
   - Use the search bar to quickly find a student

## Best Practices
- Check attendance daily to ensure records are up to date
- Use the search and filter options for quick access
- Mark holidays and absences accurately

---

## Troubleshooting
- **Attendance Not Showing?**  
  Ensure the correct class and date are selected
- **Incorrect Status?**  
  Edit the attendance record as needed
