# Monthly Attendance

The **Monthly Attendance** feature allows you to view the attendance status of each student for every day of a selected month. This grid view helps you quickly identify patterns of presence, absence, and holidays.

---

## How to View Monthly Attendance

1. **Go to:**  
   **Attendance** → **Monthly Attendance**
2. **Select Class Section:**
   - Use the dropdown to choose the relevant class section (e.g., 1 A - English)
3. **Select Month:**
   - Pick the month you want to review (e.g., January)
4. **View Attendance Grid:**
   - The table displays:
     - Roll Number
     - Student Name
     - Columns for each day of the month
     - Attendance status for each day (e.g., Present ✔️, Absent ❌, Holiday, or - for not marked)

## Best Practices
- Review monthly attendance to identify trends and address issues early
- Use the grid to spot frequent absences or unmarked days
- Communicate with students/parents about attendance patterns

---

## Troubleshooting
- **Grid Not Loading?**  
  Ensure the correct class and month are selected
- **Missing Data?**  
  Check if attendance was marked for each day
